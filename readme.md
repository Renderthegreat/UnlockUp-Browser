# UnlockUp Browser

Recently, many educational districts have been plagued by *Lockdown Browser*.
This invasive software not only is costly, but it has fundamental flaws that cannot be fixed with an amount of patches.

A secure, digital test taking environment itself is impossible.
In this demo, I exploit the fact that *Lockdown Browser* cannot tell if a `chrome` object is legitmate or not.
Simply put, I create a fake `chrome` object that acts exactly like a real one, but in reality is a mirage.
In simple terms, the extension believes it has full control, but it is actually trapped within a meticulously created sandbox I have full control of.

I didn't create this tool for cheating, and I will never stand for that. But facts are facts, and *Respondus* will eventually have to give up. No matter how many security features they add to their products, people like me will always be able to bypass it.

Not only is *Lockdown Browser* unsecure, it also punishes those who are honest. People of African race are often flagged by the AI video monitor, and others with ADHD are punished for the stress they have from being watched.

I don't wish to cause havoc, but we need to change how digital learning in a whole is viewed. My goal is to force schools to not use *Lockdown Browser* by providing this demo which allows any type of assessment to be ran within a unrestricted HTML webpage making all of the browsers security features useless. The only other option for schools would be to switch to *Windows* or *MacOS* so they can use a different browser from *Respondus*, but my hope is this will be to costly and inconvenient.
